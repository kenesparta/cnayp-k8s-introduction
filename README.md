# Cloud Native Ayacucho

## Introducción a Kubernetes: La Plataforma de Orquestación de Contenedores

Se ofrece una visión general de Kubernetes, que es una plataforma para la gestión y orquestación de contenedores.
Exploraremos sus componentes clave, su arquitectura y cómo facilita la automatización de despliegue, escalado y gestión
de aplicaciones en entornos de microservicios. Esta introducción está dirigida a quienes buscan comprender las bases
para comenzar a implementar Kubernetes en sus proyectos.
